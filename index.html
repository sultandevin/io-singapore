SINGAPORE'S SECRET WEAPON FOR CREATIVE THINKERS

This educational video game is a digital platform designed to supplement traditional learning methods by providing an immersive and entertaining educational experience. It will cover various subjects such as creativity skills, problem solving skills, memory power, and thinking process. This video game is incorporating educational content into engaging gameplay mechanics.

HERE'S THE THING... (ABOUT)

This educational video game project aims to develop an interactive and engaging video game that facilitates learning and educational experiences for children aged 6 to 12. This document outlines the key features, objectives, and specifications of the educational video game to ensure alignment with project goals requirements.

This game aims to train problem solving skills which also involve the player's memory power. At the start of the game, the user will be presented with a series of challenges that must be faced. Here the user will try to complete the challenges at each level. Then, to complete the challenge, there are several movement options available that can be done.
